# john.dev — Portfolio

Senior full-stack developer portfolio built with React, TypeScript, Vite, and Tailwind CSS v4.

## Setup

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Deploy

Deploys to GitHub Pages via `gh-pages`:

```bash
npm run deploy
```

The site is published at `https://momohbk.github.io/john-dev/`.

## Structure

```
src/
├── components/   # React components (Navbar, Hero, Skills, Projects, Contact)
├── data/         # Static data (projects, skills, tech options)
├── hooks/        # Custom hooks (useScrollPosition)
├── types/        # TypeScript type definitions
├── App.tsx       # Root component
└── main.tsx      # Entry point
```
